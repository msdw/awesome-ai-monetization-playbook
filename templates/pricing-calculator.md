## Notes on using this calculator

- LLM API costs should be estimated conservatively. Measure actual costs after the first few deliveries and update the model.
- The buyer value estimate helps frame the price conversation, not justify it internally. If you cannot articulate the value, the price will be harder to defend.
- Pricing is a hypothesis. This calculation creates a floor and a target, not a final answer.
